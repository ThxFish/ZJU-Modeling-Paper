# 建模比赛：透明塑料饮料瓶内残留液体识别

本仓库包含比赛中所要求的数据集、模型训练代码、以及 LaTeX 论文源码。

## 📁 目录结构推荐
- `data/`: 存放按 8:2 分割好的数据集图片，以及 `YOLO`/`VOC` 格式的标签。
- `code/`: 
  - `crop_classifier/`: 任务二专用（PyTorch ResNet/MobileNet 分类代码）。
  - `yolo_detector/`: 任务三专用（YOLOv8 训练脚本及推理演示）。
- `main.tex`: LaTeX 论文主文件（Overleaf 可通过 GitHub 仓库直连或传压缩包直接编译）。
- `figures/`: 存放论文中涉及到的插图和结果可视化图像。

## 📝 写作进度指导（How to continue）

### 阶段一：数据准备与任务一填充
1. **收集**：利用手机拍摄并整理网图，凑齐≥100张图片。确保覆盖各种倾斜姿态以及液体（甚至包括反光）。
2. **人工标注**：使用 `labelImg` 或 `Roboflow` 给瓶身画目标框。
3. **写论文**：在 LaTeX 第 3 节中，配上一张图展示你的数据集样例矩阵，描述包含的内容比例（例如：竖直占40%，侧放占60%）。

### 阶段二：训练二分类器，完成任务二分析
1. **预处理**：写一个 Python 脚本，根据刚画的坐标框将所有瓶子裁剪出来。
2. **模型训练**：使用 PyTorch 对剪裁的图片做 `Empty` vs `Liquid` 的二分类。
3. **写论文**：保存 Loss 曲线图、准确率趋势图和测试集混淆矩阵，插入 LaTeX 第 4 节。用恰当的文字解释（因何具有泛化能力——如：引入了数据扩增减少过拟合）。

### 阶段三：端到端 YOLO 模型，完成任务三分析
1. **修改标签**：原标签只标注了瓶身，现在给分类附上属性：`0: empty`, `1: little`, `2: medium`, `3: a_lot`。
2. **YOLO 训练**：利用 `ultralytics` 包一行代码启动 YOLOv8 训练。
3. **写论文**：在 LaTeX 第 5 节描述 YOLO 的定位原理。在论文中附上 4~6 张 YOLO 给出的预测结果图，要求涵盖各种难以识别的场景（如倒放且反光），证明定位模型能够较好工作。

### 阶段四：排版与模型评价
- 运行命令 `xelatex main.tex` 编译生成最终 pdf。
- 在第 6 节汇总本方法的优缺点，如优点：深度学习端到端、泛化好；缺点：依赖大量标注数据等。
